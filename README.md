# AI Image Template Studio

AI Image Template Studio is an early-stage open-source project for an AI-assisted image template editor.

This repository currently ships a polished static GitHub Pages demo and project roadmap. The full product direction is a React + Vite + TypeScript frontend with Supabase for auth/storage/drafts and a provider-agnostic AI Gateway for image generation workflows.

## Current status

- Static GitHub Pages demo
- Open-source roadmap
- AI Gateway design notes
- Supabase integration roadmap
- Codex for Open Source application notes
