# Codex for Open Source Application Notes

## Project summary

AI Image Template Studio is an open-source image template editor concept for creators, small businesses, and e-commerce operators. It is designed to help users create marketing visuals, social media cards, product posters, and AI-assisted image templates.

The current public release provides a polished static GitHub Pages demo, documentation, and a clear open-source roadmap. The full project direction is React, Vite, TypeScript, Supabase, and a configurable AI Gateway.

## Why Codex would help

I am the primary maintainer. I plan to use Codex to:

- Refactor the frontend architecture
- Improve documentation and onboarding
- Add tests and security checks
- Migrate platform-specific AI calls into a clean provider-agnostic gateway
- Improve Supabase setup scripts and self-hosting documentation
- Review pull requests and issues more efficiently

API credits would be used for AI Gateway testing, maintainer automation, code review workflows, documentation generation, and future AI-assisted image generation experiments within the open-source project.
